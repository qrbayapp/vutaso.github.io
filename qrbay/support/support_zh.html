打开 QRBay 应用程序，将相机对准要扫描的二维码。应用程序会自动识别并扫描代码。确保二维码光线充足且未被遮挡。

使用 QRBay 扫描产品条形码后，您可以使用快速查询功能在 Google、Bing 或 Amazon 上查找产品信息和价格。只需按下“查询”按钮，即可快速准确地搜索信息。

QRBay 支持多种格式，包括：QRCode, Code 39, Code 93, Code 128, EAN-8, EAN-13, UPC-A, UPC-E, ITF, Data Matrix, AZTec, PDF 417 和 Codabar。

扫描代码后，您可以将结果保存到应用程序的库中，或通过电子邮件、短信或其他社交媒体应用程序直接分享。共享功能位于扫描结果页面的右上角。

QRBay 可用于课堂上创建和扫描作业、参考资料和在线测验的二维码。教师可以为学习链接创建二维码，学生可以快速扫描以访问它们。这有助于优化时间并提高学习兴趣。

QRBay 承诺保护用户隐私。未经您的同意，应用程序不会收集或存储任何个人数据。扫描数据仅存储在您的设备上，除非您选择将其外部分享。

您可以通过设备的应用商店（Android 使用 Google Play 商店，iOS 使用 App Store）更新 QRBay 应用程序。建议您将应用程序更新到最新版本，以享受最新功能和改进。

QRBay 支持多种语言以服务全球用户。您可以在设备的设置中更改语言以满足您的需求。

QRBay 支持多达 18 种语言，包括：英语、越南语、印尼语、中文、泰语、韩语、日语、法语、德语、西班牙语、波兰语、土耳其语、俄语、葡萄牙语、意大利语、荷兰语、阿拉伯语和印地语。

QRBay 允许您直接从设备的相册扫描二维码。在应用程序中选择“从相册扫描”选项，然后选择包含要扫描的二维码的图像。

是的，QRBay 提供创建具有自定义设计和信息的二维码和条码的功能。您可以输入要编码的内容并选择自定义设计以符合您的喜好。

QRBay 应用程序会记录您的扫描和代码创建历史记录，以便您轻松访问和管理。您可以在应用程序的“历史记录”部分中找到历史记录，并执行查看、分享或删除等操作。

您可以轻松地 在 Facebook、Twitter、Instagram 等社交媒体平台上分享扫描或创建的二维码，只需点击几下即可。共享功能位于二维码详细信息部分。

通过 QRBay 的批量扫描功能，您可以一次扫描多个二维码和条形码，节省时间并提高效率。选择批量扫描模式，将相机对准代码即可开始扫描。

QRBay 允许您以 CSV、文本或图像格式导出扫描数据，以便轻松分析和分享。导出数据选项可在应用程序的扫描历史记录部分中找到。
